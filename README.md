# Dursa Jemal Mohammed - Accounting & Finance Portfolio

A modern, responsive, interactive portfolio website and financial tools suite for **Dursa Jemal Mohammed**, BA Graduate in Accounting & Finance from Harambee University.

![Portfolio Preview](profile.jpg)

## 🌟 Overview

This web application serves as a comprehensive professional portfolio showcasing background, skill set, academic achievements, and practical financial tools. Built with high performance, elegant UI/UX, glassmorphism design effects, and native dark/light theme support.

---

## ✨ Features

- **Professional Bio & Credentials**: Highlights education (CGPA 3.4, Exit Exam Score 69%), core competencies, and career objectives in Auditing, Banking, and Financial Strategy.
- **Interactive Financial Tools**:
  - 🧮 **Ethiopian Income Tax Calculator**: Calculates taxable income, progressive tax brackets, employee pension deductions, and net salary.
  - 📊 **Loan Amortization Generator**: Computes monthly payments, total interest, and produces monthly repayment schedules.
  - 📈 **Financial Ratio Analysis**: Analyzes Current Ratio, Debt-to-Equity Ratio, Net Profit Margin, and Return on Equity (ROE).
  - 📦 **Inventory Valuation Simulator**: Compares FIFO, LIFO, and Weighted Average inventory costing methods.
- **Resume Download**: Direct link to download Dursa Jemal's official CV/Resume (`Dursa CV.pdf`).
- **Interactive UI**: Smooth scrolling navigation, dynamic theme switching (Dark/Light mode), dynamic counters, and responsive design for all mobile & desktop screens.

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom CSS Grid/Flexbox, Glassmorphism, CSS Variables), JavaScript (ES6+)
- **Icons & Fonts**: Font Awesome 6, Google Fonts (*Inter* & *Outfit*)
- **Deployment Ready**: Standard static site structure compatible with GitHub Pages, Vercel, Netlify, or any HTTP server.

1. Clone or download this repository:
